Online Quiz Management System
  




In this Online Quiz system the users created by the user having the admin role, can login through their username and password which was given by the admin during the adding process. When the user login through the username and password according to their role given they got redirected to their respective user pannels.

And if they give something else then their given username and password then a eror sweet alert popped in front of them of invalid credential.



If an admin user login to our system by putting his/her given credential then upon successfull login this sweet alert is popped that you are going to redirect to the admin panel or Dashboard.

And if the given username and password does not exits in the database then a eror sweet alert popped in front of them of invalid credential.



If any of the user tries to login with wrong credential (i.e., with wrong username or wrong password) then the shown sweet alert popped that your entered credential is invalid. This also popped when you entered a credential which is not there in database or removed by the admin user then also this sweet alert is popped on the screen.



After a user gets logged as admin the shown dashboard is visible to him/her where he can do several things like to delete any question or quiz and add user by assigning their role and also can able to delete them and also he/she can modify his/her profile.
It has a navbar to navigate to different section to see the students and faculty connected and can also the question and quiz created. In home section he/she has a greeting from the system according to the login time and a overall data of student, faculty, question and quizzes created or conneted.



This is the section where the user with the admin role can add an user by filling the fields like the email, username, password and seleting the role for the creating user then submitting the form. Then upon successfully added the newly created user, a sweet alert is popped on the screen displaying the role of the user and an assertive message for the addition of the user.



The user can also edit his/her profile and can change the credential from this section. He/She can change the email, username, password. On clicking on the submit button the changed data gets updated in the system.

But he/she cannot his role even if he/she has the admin role as by himself/herself he/she can't change the role of his/her own. But the other admin can chage the role of the other.



If an faculty user login to our system by putting his/her given credential then upon successfull login this sweet alert is popped that you are going to redirect to the faculty panel or Dashboard.

And if the given username and password does not exits in the database then a eror sweet alert popped in front of them of invalid credential.



After a user gets logged as faculty the shown dashboard is visible to him/her where he can do several things like to delete or add or edit any question and can generate quiz or edit any user and can chage their password, email and password.
It has a navbar to navigate to different section to see the students connected and can also the question created and to generate any new quiz. In home section he/she has a greeting from the system according to the login time and a overall data of student, question and quizzes created or conneted.



This is the section where the faculty user has the power to add the question to the system through this window or section.

There are several input fields where he/she can type the question and can give options for the question typed and assign the question with the tecnology. And he/she can select the time he/she want to be given to solve the question. And then on click on the add question button the question gets addeed to the system repository, which is afterward taken use to generate quizzes.



This section gives the faculty users with the power to generate quizzes, which can be later being used to refer to the users who can give tests for the generated quizzes by the faculty user.

This contains two fiels one to assign the questions with one quiz name and other to classify the quiz to be generated with the technologies and on clicking the generate quiz button the system randomly picks 5 questions from the technology mentioned and generate a quiz of these 5 question and named it with the name given by the user. So, that the user can give the test for this by selecting the name of the quiz.



If an student user login to our system by putting his/her given credential then upon successfull login this sweet alert is popped that you are going to redirect to the student panel or Dashboard.

And if the given username and password does not exits in the database then a eror sweet alert popped in front of them of invalid credential.



After the user logged in with the student role then he/she gets redirected to the shown panell or dashboard where the navbar contains a logo to specify the stdent dashboard and a logout button and in the middle of the dashboard there are three options, the first for given the quiz tests and second for seeing his/her performances in different technologies and the third one for practising the questions from the question bank.



After clicking on the "solve Quizz" tab there comes a screen with a feild having options to select technology for which you want to solve the quizzes. On selecting the technology there comes another giving options to select the quizz by selecting the quizname.
Upon selecting the technology and quiz there comes the questions inside the selected quiz one by one along with the timer.We can give the answer by selecting the radiobutton in front of each option, then click on the submit button, then the question with the answer gets submitted and it moves to the next question of the quiz. After the completion of the timer the question gets autosubmitted and it moves to the next question of the quiz.



After successfully submition of all the questions of the quiz the system provides a quick result based on the test that out of the total question, how many you do correct. On which you can get the percentage of your performance. It also give a greeting for the solving of the quizzes on the system.



After clicking the "view performance" tab of the student dashboard the user gets redirected to the shown section, where he/she can see his/her performance in different technologies.

Here, each performance of the user in different technologies is depicted in each bars of the graph and a tooltip is provided giving the information about the technology and percentage of performance out of 100 in that technology on hover on the bar.



After clicking the "Quiz Bank" the user gets redirected to a section where the user can see all the questions present inside the system in the format as shown. And also he can practice from the question as on the click on the options, if choosen option is the correct option then its bg changes to green or otherwise it changes to red depicting that the chosen option is incorrect. There is an option of "View Answer" at the last of each question which contains the correct option of each question, which bcomes visible upon clicking on the button "View Answer".
