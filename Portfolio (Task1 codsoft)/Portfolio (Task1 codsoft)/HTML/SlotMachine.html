Slot Machine Game
 
 
In this website, I tried to create a virtual and interactive slot machine which has a screen for the scroll of the objects and another for the result display in the top of the object screen, where it shows a "Not Matched", when three of the objects in the object screen is not same. You can make another roll by clicking the lever next to the object screen.
When the three objects, in the object screen depicts the same object as shown then you gets a jackpot. Now the result screen will written with the "Jackpot" and the below the logo "saayan samanta" will turn green from red.
