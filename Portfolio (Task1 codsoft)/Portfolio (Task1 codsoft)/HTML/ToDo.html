ToDo List Website
 
 


This is a website which helps you to store or note down the todos in a systematic and serial manner and also you can delete or remove the todo which you have completed.


In this website there is a field, where you can type the todo and then after clicking on the plus icon next to the input feild, the inputed todo gets added to the todo list with the proper serial number and delete icon which is used to delete the perticular todo.


After adding the elements in the todo list, you can remove the elements from the list by simply clicking on the bin icon in front of each element of the list. By clicking on the icon the corresponding element gets deleted from the list and the serial numbering of the rest of the elements gets organised accordingly automatically.
